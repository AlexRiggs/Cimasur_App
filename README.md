Panel Administrativo CIMASUR

Sistema de gestión para condominios y residencias CIMASUR.

Características

Gestión de Roles:

Admin: Control total de pagos y usuarios.

Mesa Directiva: Panel de auditoría (solo lectura).

Residente: Consulta de adeudos y reserva de áreas comunes.

Base de Datos: Integración en tiempo real con Supabase.

Pagos: Registro y visualización de estados de cuenta.

Reservas: Sistema para apartar áreas comunes (Palapa, Tenis, Salón).

Tecnologías

React + Vite

Tailwind CSS

Lucide React (Iconos)

Supabase (Backend & Auth)

Instalación

Clonar el repositorio.

Ejecutar npm install.

Configurar las variables de entorno en .env.

Ejecutar npm run dev.
